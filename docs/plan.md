# Development Plan

This implementation follows `智者科技新闻智能体计划书 V1.1` and completes the first-week MVP scope:

1. Multi-agent skeleton and orchestrator.
2. Data models and configuration files.
3. HN, RSS, arXiv, and GitHub collectors.
4. Cleaning, URL normalization, and lightweight clustering.
5. Rule-based classification and scoring.
6. Extractive summary fallback with optional LLM client.
7. Markdown briefing generation and scheduled GitHub Actions run.
